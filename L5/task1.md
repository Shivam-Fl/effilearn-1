1.	Open your profile-page folder from Lecture 2.
2.	Create a new file in the same folder named style.css.
3.	Link the stylesheet inside profile.html’s <head>: <link rel="stylesheet" href="style.css">
4.	In style.css, style the body: set a font-family and a background-color.
5.	Style the <h1>: pick a color and center it with text-align: center.
6.	Style the <img>: center it with display: block; and margin: 0 auto; then try border-radius: 50% to make it circular.
7.	Style the <table>: add border-collapse: collapse on the table, and padding plus a border on th and td.
8.	Save both files and refresh your browser to see the changes.
