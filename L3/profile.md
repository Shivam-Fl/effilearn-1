1.	Create a new folder named my-profile-page (or reuse your folder from Lecture 1).
2.	Inside it, create a new file named profile.html.
3.	Type the basic HTML boilerplate (<!DOCTYPE html>, <html>, <head>, <body>).
4.	Add an <h1> with your name.
5.	Add an <img> with a photo. If you don’t have one handy, use a placeholder: src="https://placehold.co/150".
6.	Add a <p> introducing yourself in two or three sentences.
7.	Add an <h2> “My Hobbies” followed by a <ul> with a few <li> items.
8.	Add an <h2> “Useful Links” followed by two or three <a> links.
9.	Save the file and open it in your browser to check the result.
