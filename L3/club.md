1.	Create a new file named club.html inside your project folder.
2.	Add the page boilerplate and an <h1> with a club name of your choice.
3.	Add an intro <p>, then an <h2> “Why Join?” with a <ul> of three benefits.
4.	Add an <h2> “Membership Plans” with a <table> comparing at least two plans.
5.	Add an <h2> “Sign Up” with a <form> containing: a text input for name, an email input, radio buttons for experience level, checkboxes for interests, a textarea for comments, and a submit button.
6.	Save the file and open it in your browser. Check that each radio-button group only lets you select one option.
